# interview machine learning engineer
* How would you explain to a non-engineer(like a client) what is the value of machine learning?
* Explain the whole process of building a machine learning project. What stages are necessary?
* Which algorithms have you worked with before?
* You have the data. You have the algorithms. How will you find the best model and evaluate it? 
* What does a machine learning engineer do?
* What are some other things that machine learning engineers do?
* Why did you choose to become a machine learning engineer as a career?
* How do machine learning engineers work with other developers?
* What are some challenges of being a machine learning engineer?
